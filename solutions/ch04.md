## 演習 1: RoPE スコア行列の可視化

### ヒント
同じ $q = k$ を複数の位置に並べ、`apply_rope` で位置ごとに回転させる。
内積行列を heatmap にすれば「対角から離れるほど減衰」が見える。

### 完全解
[`examples/ch04/rope_demo.py`](../examples/ch04/rope_demo.py)

matplotlib を加えるなら:

```python
import matplotlib.pyplot as plt
plt.imshow(matrix(True).detach(), cmap="RdBu_r")
plt.colorbar(); plt.title("With RoPE"); plt.show()
```

## 演習 2: RoPE なしの比較

`apply_rope` を挟まなければ、同じベクトルの内積はすべて等しく、
**行列全体がフラット** になる。これが「位置情報が無い」状態。

## 演習 3: `LlamaAttention` の対応

`transformers/models/llama/modeling_llama.py` の中で:

- `apply_rotary_pos_emb` ← 本章の `apply_rope`
- `_get_rotary_emb` / `LlamaRotaryEmbedding` ← `rope_cache`
- `LlamaAttention.forward` 内の

  ```python
  query_states = apply_rotary_pos_emb(query_states, cos, sin, position_ids)
  key_states   = apply_rotary_pos_emb(key_states,   cos, sin, position_ids)
  ```

  が本章の **Q, K にのみ RoPE を適用** に対応する。
