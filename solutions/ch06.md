## 演習 2: KL 距離の計算

離散分布 $p, q$ の KL は

$$
\mathrm{KL}(p \| q) = \sum_i p_i \log (p_i / q_i)
$$

例: $p=[0.5, 0.5], q_1=[0.6, 0.4], q_2=[0.95, 0.05]$。

```python
import torch
p = torch.tensor([0.5, 0.5])
for q in (torch.tensor([0.6, 0.4]), torch.tensor([0.95, 0.05])):
    kl = (p * (p.log() - q.log())).sum()
    print(kl.item())
# => 0.020 / 1.33
```

β = 0.02 ならほぼ動けない、β = 2.0 なら大きく動けるが壊れやすい。
