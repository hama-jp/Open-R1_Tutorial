## 演習 2: FFN を GELU版 と SwiGLU版 で比較

### ヒント
- GELU 版: `Linear(d, d_ff) → GELU → Linear(d_ff, d)` の 2 層
- SwiGLU 版: 上記 + もう 1 本のゲート用 Linear(d, d_ff)
- パラメータ数は SwiGLU が $\sim 1.5\times$（厳密には $3 d d_{ff} / 2 d d_{ff} = 1.5$）

### 部分解

```python
class GELUFFN(nn.Module):
    def __init__(self, d, d_ff):
        super().__init__()
        self.w1 = nn.Linear(d, d_ff, bias=False)
        self.w2 = nn.Linear(d_ff, d, bias=False)
    def forward(self, x): return self.w2(F.gelu(self.w1(x)))

print("GELU   :", sum(p.numel() for p in GELUFFN(64, 128).parameters()))
print("SwiGLU :", sum(p.numel() for p in SwiGLU(64, 128).parameters()))
```

出力例:
```
GELU   : 16384
SwiGLU : 24576
```
